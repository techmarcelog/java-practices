# java-practices

Java practice repository focused on data structures.

## Structure

Each topic includes:
- Java implementation in `src/main/java/com/javapractices/datastructures/...`
- Topic-specific documentation in `docs/<topic>/README.md`

## Included topics

- Pointers (Java references)
- Lists
- Linked Lists
- Hash Map
- Set and Maps
- Queue
- Stack
